Git Setup Plan

1. Verify whether Git is installed on Windows.
   - Open PowerShell and run `git --version`.

2. If Git is not installed, download and install Git for Windows.
   - Visit https://git-scm.com/download/win
   - Choose the option to use Git from the command line and 3rd-party software.

3. Restart the terminal or VS Code after installation.

4. Confirm installation.
   - Run `git --version` again.

5. Initialize the repository in the project folder.
   - Run `git init` from `C:\Users\subik\OneDrive\Desktop\portfolioo\portfolio`.

6. Add project files and make the initial commit.
   - `git add .`
   - `git commit -m "Initial commit"`
